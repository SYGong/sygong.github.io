## Solution in Python3
```python
class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        num_i = {}  # Number to index
        for i, v in enumerate(nums):
            complement = target - v
            if complement in num_i:
                return [num_i[complement], i]
            num_i[v] = i
```
